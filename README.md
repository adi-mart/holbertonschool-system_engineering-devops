# holbertonschool-system_engineering-devops

This repository contains projects and exercises related to system engineering and DevOps, as part of the Holberton School curriculum. Each directory covers a specific topic or set of tasks designed to build practical skills in infrastructure, automation, and operations.

## Main Directories

- **web_infrastructure_design/**: Contains tasks and diagrams for designing scalable, secure, and monitored web infrastructures. Each task is explained in English and includes a visual diagram. See the README in this directory for detailed explanations of each infrastructure scenario.
